Movie Watchlist Site

This movie site grabs movies based on titles inputted into the search. 
It accesses https://www.omdbapi.com to get information based on search and 
return movies based on said result.

If you press the watchlist button on any search result it will save that result to your 
watchlist which save to your browsers local storage. 


******API KEY****
In index.js, there is a variable named "apiKey". 
You will need to get your own api key from: https://www.omdbapi.com/apikey.aspx once 
you have the api key insert it into the variable and the application shall work.
